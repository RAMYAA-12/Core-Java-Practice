# Core Java Practice

This repository contains basic Java programs created
while practicing core Java concepts and logic building
for campus placements and technical interviews.

## Topics Covered
- Basic arithmetic operations
- Number-based problems (MSB, unit digit, sum in range)
- Conditional statements
- Loops and simple logic problems

## Technology Used
- Java 8
- Eclipse IDE

## Purpose
This repository is maintained with daily practice
to strengthen problem-solving and core Java fundamentals.
